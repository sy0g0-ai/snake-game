# Snake Game

シンプルで楽しいスネークゲームです。PC・スマホ両対応、ブラウザだけで動作します。

## プレイ方法

`index.html` をブラウザで開くだけでプレイできます。サーバーやインストールは不要です。

## 操作

| 環境 | 操作方法 |
|------|----------|
| PC | 矢印キー（↑ ↓ ← →）または W / A / S / D |
| スマホ・タブレット | 画面をスワイプ または 画面下のDパッドボタン |

## ゲームの特徴

- **スコア・ベストスコア・レベル** の表示（ベストスコアはブラウザに自動保存）
- **レベルアップシステム** — スコアに応じてスピードが上がる
- **ボーナスアイテム（★）** — ランダムに出現し、時間内に取ると3倍点
- **完全オフライン動作** — HTMLファイル1つで完結
- **レスポンシブデザイン** — スマホ〜PC まで画面サイズに自動対応

## ファイル構成

```
snake-game/
└── index.html   # ゲーム本体（HTML / CSS / JavaScript すべて内包）
```

## 技術スタック

- HTML5 Canvas
- Vanilla JavaScript（フレームワーク不使用）
